# Hue

一款基于Unity开发的2D探索解谜游戏，玩家需通过观察材质混乱规律，使用炸弹清除异常状态，还原场景正常秩序。


## 项目信息
**游戏类型**：2D探索解谜（材质修复主题）  
**开发引擎**：Unity 2022.3.62f2c1  
**开发模式**：团队开发  
**开发周期**：2025.12.01 - 2025.12.15

## 核心玩法
### 操作方式
移动：WASD 控制角色上下左右移动  
交互：鼠标点击投掷炸弹

### 核心逻辑
1. 场景内材质呈现随机混乱状态，需观察并识别材质排列规律
2. 使用炸弹精准引爆异常材质区域
3. 清除当前场景所有混乱材质后过渡到新关卡

### 通关目标
清除场景内全部异常材质，过渡到新关卡完成全关卡挑战。

## 运行说明
### 环境要求
操作系统：Windows 10+ / macOS 12+（64位）  
Unity版本：2022.3.62f2c1（必须匹配此版本避免兼容问题）

### 启动步骤
1. 克隆仓库
   ```bash
   git clone https://github.com/ggbond534/Hue.git
2.导入项目：打开 Unity Hub → 添加项目 → 选择克隆后的 Hue 文件夹  
3.等待资源加载完成（首次加载可能需要 5-10 分钟）  
4.运行游戏：
在 Project 窗口中打开 Scenes/Hue.unity  
点击 Unity 编辑器顶部的播放按钮（▶️）启动游戏
