# Hue

这是一个基于C#及Unity实现的一个2D小游戏，玩家通过寻找材质混乱规律，使用炸弹清除异常状态，还原场景正常秩序的2D探索解谜游戏

核心信息
游戏类型：2D探索解谜（材质修复主题）
开发引擎：Unity 2022.3.62f2c1
开发信息：团队开发 | 开发周期：2025.12.01 - 2025.12.15
 
玩法介绍
1. 操作：WASD控制角色移动，鼠标点击投掷炸弹；
2. 核心逻辑：场景内材质呈随机混乱状态，需观察材质排列规律，用炸弹精准引爆异常材质；
3. 通关目标：清除当前场景所有混乱材质，解锁新关卡，解锁不同威力的炸弹道具。
 
功能模块
核心玩法：材质混乱生成逻辑、炸弹引爆判定系统、关卡解锁机制、道具属性区分；
技术特性：角色移动与碰撞检测、炸弹爆炸特效触发、场景状态管理；
拓展功能：多类型炸弹道具（范围/精准/延时）、关卡难度梯度设计、场景材质视觉区分效果。
 
运行说明
 
环境要求
运行环境：Windows 10+/macOS 12+（64位系统）
开发依赖：Unity 2022.3.62f2c1
 
启动方式
1. 克隆仓库： git clone https://github.com/ggbond534/Hue.git 
2. 打开项目：用Unity 2022.3.62f2c1导入克隆后的项目文件夹，等待资源加载完成；
3. 运行游戏：打开Scenes文件夹下的 Hue.unity ，点击Unity编辑器播放按钮即可启动。
